## 👉 The cheatsheet

**[MARKDOWN_CHEATSHEET.md](./MARKDOWN_CHEATSHEET.md)** — every piece of syntax you'll forget at 2 a.m., in one scroll.

## 🐈‍⬛ Why this exists

I kept re-Googling the same five markdown things. Then I kept re-Googling the same *other* five. Eventually I wrote everything down so I could stop.

This repo is that writeup, shared publicly in case it saves someone else a browser tab.

## 📑 What's covered

- Basic syntax (headers, text, lists, links, images, code, tables, quotes)
- GitHub-specific features (alerts, collapsibles, task lists, mermaid, LaTeX)
- Badges, anchors, and HTML-in-markdown
- README patterns worth stealing
- Gotchas that will bite you eventually

## 🛠️ Use it

Clone, fork, copy-paste, or just bookmark. No install, no dependencies — it's a markdown file.

```bash
git clone https://github.com/cfoudysec/markdown-cheatsheet.git
```
